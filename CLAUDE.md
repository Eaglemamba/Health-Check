# Health-Check

個人健康追蹤系統，使用 Markdown 檔案記錄每日健康數據並定期回顧。

## 專案結構

```
health-check-daily-setup/   # 模板與指令
  _template.md              # 每日紀錄模板
  _weekly-template.md       # 每週回顧模板
  daily-command.md          # 每日 check-in 流程指令
reviews/
  daily/                    # 每日紀錄（YYYY-MM-DD.md）
  weekly/                   # 每週回顧（YYYY-Wxx.md）
  monthly/                  # 每月回顧（YYYY-MM.md），模板在 _template.md
```

## 每日 Check-in 流程

執行 `daily-command.md` 中的流程，逐項引導填寫：體重、血壓、睡眠、身體信號、今日計畫。

填寫完成後，同步將當天數據更新至對應的 weekly review 檔案。若該週檔案不存在，從 `_weekly-template.md` 建立。

## 重要規則

- 語言：所有健康紀錄與模板使用**繁體中文**
- 每次 check-in 控制在 3 分鐘內，簡潔不討論
- 體重：週日早晨為正式紀錄（排尿後、進食前）
- 血壓：連量兩次取第二次；收縮壓 > 160 或 < 90 提醒就醫
- Sleep Score 連續三天 < 65 → 提醒檢視睡眠修復方案
- 身體信號連續三天非「清」→ 提醒關注趨勢

## 檔案命名慣例

- Daily: `reviews/daily/YYYY-MM-DD.md`
- Weekly: `reviews/weekly/YYYY-Wxx.md`
- Monthly: `reviews/monthly/YYYY-MM.md`

## 物理治療 & 健康管理建議規則

每日 check-in 完成記錄後，根據以下規則**自動提供簡短建議**（2-3 句，不打斷 3 分鐘節奏）。建議以「💡 PT 建議：」開頭，附在紀錄尾端。

### 一、身體信號觸發建議

依據回報的**部位 + 性質**，提供對應的物理治療自我照護建議：

| 部位 | 常見性質 | 建議方向 |
|------|----------|----------|
| 前臂 / 手腕 | 酸緊、僵硬 | 前臂伸展（手掌朝上，另一手輕壓手指向下 30 秒 ×2）；工作每 50 分鐘休息活動手腕 |
| 肩胛骨 | 酸緊、沉重 | 肩胛骨收放（Y-T-W 靠牆動作 8 下 ×2 組）；檢查螢幕高度是否低於視線 |
| 頸部 | 僵硬、緊繃 | 頸部側彎伸展 30 秒/側；收下巴運動 10 下（改善前頭姿勢） |
| 下背 | 痠痛、僵硬 | 貓牛式 8 下 ×2；避免久坐超過 45 分鐘；坐姿時用腰靠 |
| 膝蓋 | 痠痛 | 確認靠牆深蹲角度是否太深，退回上一階段；髕骨活動術（坐姿推髕骨上下左右） |
| 肩膀 | 痠痛、活動度受限 | 肩關節鐘擺運動；胸椎旋轉活動度練習；暫停伏地挺身相關動作 |
| 髖部 | 緊繃 | 90/90 髖關節活動度練習；半跪姿髂腰肌伸展 30 秒/側 |
| 足底 | 疼痛 | 凍水瓶踩滾 2 分鐘；小腿拉伸（弓箭步靠牆）30 秒 ×2/側 |

**通用規則：**
- 同一部位**強度 ≥ 5** → 建議當天該部位避免高負荷訓練，改做輕度活動
- 同一部位**強度 ≥ 7** → 建議暫停相關運動，48 小時觀察，未改善就預約物理治療
- 同一部位**連續 3 天非清** → 提供該部位的完整居家復健菜單（3-4 個動作）
- 同一部位**連續 5 天非清或強度持續 ≥ 5** → 強烈建議預約物理治療師評估

### 二、數據異常觸發建議

比對前日或本週趨勢，若出現以下異常自動提示：

**體重：**
- 單日波動 > 1 kg → 提醒可能是水分/鈉攝取變化，非脂肪變化，不必緊張
- 一週增加 > 0.5 kg → 檢視本週飲食和飲水執行率，是否有外食增加
- 一週減少 > 1.5 kg → 警告太快，有痛風發作和膽結石風險，建議放慢

**血壓：**
- 收縮壓比前日上升 > 15 mmHg → 確認是否昨晚睡眠不足、壓力大、或攝入高鈉食物
- 收縮壓連續 3 天 > 140 → 建議回診諮詢醫師
- 收縮壓 < 100 → 確認是否頭暈，注意起身速度，增加飲水
- 心率突然上升 > 15 bpm（相比本週均值）→ 確認是否生病、脫水、或壓力過大

**睡眠：**
- Sleep Score < 65 → 提醒當天運動強度降一級（RPE 不超過 4/10），優先恢復
- Sleep Score < 65 + Body Battery < 40 → 建議當天只做輕度伸展，暫停靠牆深蹲和快走
- Body Battery 連續 3 天 < 50 → 提醒：你的恢復力嚴重不足，檢視睡眠修復方案中的三個要點（飲水截止 17:00、睡前抬腳、5:30 前不離床）
- 主觀恢復感 ≤ 2 → 今天以恢復為主，建議做 10 分鐘溫和伸展取代原定運動計畫
- 主觀恢復感與 Garmin 數據嚴重不符（例如 Score 80 但主觀 2）→ 提醒留意是否有心理壓力、情緒因素

**飲水：**
- 實際飲水 < 1.5L → 提醒尿酸排泄效率降低，明天優先補回飲水習慣
- 連續 3 天 < 2.0L → 強調飲水不足對血壓和尿酸的雙重影響

### 三、運動安全與調整建議

根據當天身體狀態，自動調整運動建議：

- **恢復充足**（Sleep Score ≥ 70 + 身體信號「清」+ 主觀 ≥ 4）→ 可按計畫執行或嘗試進階
- **恢復一般**（Sleep Score 65-70 或身體信號 1-2 處輕微）→ 按計畫執行，不進階
- **恢復不足**（Sleep Score < 65 或 Body Battery < 40 或主觀 ≤ 2）→ 降級：靠牆深蹲減一組或減 10 秒/組；快走改散步
- **有疼痛信號**（任何部位強度 ≥ 5）→ 避免涉及該部位的動作，提供替代方案

**靠牆深蹲特別注意：**
- 膝蓋有任何不適 → 退回上一階段角度，確認姿勢六要點
- 做完後膝蓋痛持續 > 2 小時 → 暫停 3 天，重新從更淺角度開始
- 做的時候頭暈或胸悶 → 立即停止，可能是憋氣或血壓問題

### 四、模式識別（週回顧時使用）

週回顧時自動分析以下模式，並在「本週觀察」提供摘要：

- **上交叉症候群模式：** 前臂 + 手腕 + 肩胛骨 + 頸部反覆出現 → 建議每日胸椎活動度 + 肩胛穩定訓練，檢視工作站人體工學
- **下肢負荷模式：** 膝蓋 + 髖部 + 足底反覆出現 → 檢視靠牆深蹲進度是否太激進，是否需要加入暖身
- **恢復不足模式：** Sleep Score 週均 < 65 + 身體信號頻繁 → 優先處理睡眠，本週運動降級
- **壓力累積模式：** 血壓上升趨勢 + 心率上升 + Sleep Score 下降 → 提醒關注壓力源，建議增加呼吸練習（4-7-8 呼吸法）
- **正向進步模式：** 指標穩定改善 → 正面肯定，可考慮按計畫進階到下一階段
