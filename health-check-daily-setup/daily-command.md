開始今天的晨間健康紀錄。

1. 在 `reviews/daily/` 建立今日檔案：`YYYY-MM-DD.md`
2. 複製 `reviews/daily/_template.md` 的內容
3. 將 `[DATE]` 替換為今天的日期
4. 逐項引導我填寫，一次一個問題：
   - 體重（可選填，週日為正式紀錄日）
   - 血壓：收縮壓 / 舒張壓 / 心率（兩次測量）
   - 睡眠：就寢時間、起床時間、中途醒來、Garmin Sleep Score、Body Battery
   - 身體信號：任何張力、疼痛、疲勞，或回答「清」
   - 今日計畫：飲水、補充品、運動安排
5. 每回答完一題，立即寫入檔案
6. 同步更新 `reviews/weekly/` 對應週次檔案（如 `2026-W12.md`）：
   - 將今日體重填入體重表格對應欄位
   - 將第二次血壓讀數填入血壓表格對應日期列
   - 將睡眠數據填入睡眠表格對應日期列
   - 將身體信號更新至「身體信號模式」區塊（新增或累加出現天數）
   - 將運動內容填入「本週運動紀錄」表格
   - 如果該週的 weekly 檔案不存在，先從模板建立
7. 控制在 3 分鐘內完成 — 簡潔，不討論

注意事項：
- 如果血壓收縮壓 > 160 或 < 90，提醒就醫
- 如果 Sleep Score 連續三天 < 65，提醒檢視睡眠修復方案
- 如果身體信號連續三天非「清」，提醒關注趨勢
